# IP address tracker

## Project Description! 👋

A web app that shows your IP address location on a map and lets you search for any IP or domain.

## Features

- View your current IP location
- Search any IP or domain
- Interactive map display
- Responsive design

## Live Demo

https://ipaddress-tracker-app.netlify.app

## Tech Stack

- HTML
- CSS
- JavaScript
- LeafletJS
- IP Geolocation API

## Installation

1. Clone the repo
   git clone https://github.com/jyotiboniwal/ip-address-tracker.git

2. Open the folder
   cd ip-address-tracker

3. Open index.html in browser

## API Used

- IPify Geolocation API
